HDPS

HDPS - Heart Disease Prediction System is an android application which helps user to know about their chances to get heart stroke in future. The application has simple interface where user enter their details and application will process their details in algorithm which shows how much percent they have chance to get Heart Stroke. The Algorithm used in this application is MLP (Multi Layer Perceptron) Algorithm. Click here to know how algorithm works. MLP ALGORITHM. Cleveland Datasets from UCI Repository had been used to train the algorithm.

To create MLP Algorithm i have used DeepLearning Library for java. Through which it was easier for me to build MLP Algorithm in java.
